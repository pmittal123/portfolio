# 🧑‍💻 Priyanshu Mittal – Personal Portfolio

A fully responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**.  
It showcases my projects, skills, and sections like a personalized intro, featured work, and possibly a creative "pese" (poem or personal message) section.

> ✅ Designed and built from scratch as a learning and self-branding project.

---

## 📌 Features

- Clean and modern landing page (`index.html`)
- Projects section powered by `portfolio.js` and styled with `portfolio.css`
- Custom interactive section in `pese.html` with `pese.css`
- Image assets stored in the `img/` directory
- Mobile-responsive design

---

## 📁 File Structure

Portfolio/  
├── index.html # Main homepage    
├── portfolio.css # Styles for project showcase  
├── portfolio.js # JavaScript for dynamic project display  
│  
├── pese.html # Personal or creative section (e.g. poem)  
├── pese.css # Styling for 'pese' section  
│  
├── img/ # Folder containing images and assets  
└── README.md # Project documentation  



---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Layouts, animations, responsiveness
- **JavaScript** – Interactive elements and project loading
